# 🚀 Inicio Rápido - Ojo Rojo React App

## Comandos Esenciales

### 1. Instalar (solo la primera vez)
```bash
cd "ojo red/react-app"
npm install
```

### 2. Ejecutar en desarrollo
```bash
npm run dev
```

### 3. Abrir en navegador
```
http://localhost:3000
```

## ✅ Verificación Rápida

Después de `npm install`, todos los errores de TypeScript desaparecerán automáticamente.

## 📁 Estructura

```
react-app/
├── app/              → Páginas de Next.js
├── components/       → Componentes React
├── public/           → Imágenes y assets
└── [configs]         → Archivos de configuración
```

## 🐛 Problemas Comunes

### Error: Cannot find module 'react'
**Solución**: Ejecutar `npm install`

### Puerto 3000 en uso
**Solución**: `npm run dev -- -p 3001`

### Errores de TypeScript
**Solución**: Los errores se resuelven con `npm install`
